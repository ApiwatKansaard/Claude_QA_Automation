# Selector Maps

This directory contains JSON selector maps generated by the `playwright-automator` agent
when inspecting application URLs via `/auto:inspect`.

## Format

Each file maps page elements to their selectors:

```json
{
  "dashboard": {
    "jobList": {
      "selector": "[data-testid='job-list']",
      "type": "testid",
      "description": "Main job list table"
    },
    "createButton": {
      "selector": "button:has-text('Create')",
      "type": "text",
      "description": "Create new job button"
    }
  }
}
```

## Files

Selector maps are organized per feature/page:
- `scheduled-jobs-dashboard.json` — Dashboard page selectors
- `scheduled-jobs-create.json` — Create wizard selectors
- etc.

These files are **generated by the agent** — edit manually only when correcting outdated selectors.
